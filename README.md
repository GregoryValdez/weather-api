# Weather-Api
I used an open source API from [Open-Meteo](https://open-meteo.com) to create a site that accesses endpoints. My site accesses the current weather and a 7 day forecast for where I live.

## Table of Contents
- [Installation](#installation)
- [Usage](#usage)
- [Contributing](#contributing)
- [Screenshot](#screenshot)

## Installation
1. Clone the repository:
```bash
 git clone https://github.com/GregoryValdez/weather-api.git
```

## Usage
Just click the navigation links in order to display each endpoint accessed.
